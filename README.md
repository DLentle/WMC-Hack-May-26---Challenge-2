# WMC-Hack-May-26---Challenge-2

GP Practice to Clusters: 
A lookup table for GP practice code/ID, GP cluster and health board. (https://wrds.wales.nhs.uk/Search)

PCC_Complete_Data: 
The complete data extract from the Public Health Wales Primary Care Clusters Dashboard.
Table 1 in this excel file contains cluster populations for 2024 and table 11 contains population by deprivation. (https://publichealthwales.shinyapps.io/PCC_Dashboard_Eng_Accessible/)

Disease registers by local health board, cluster and GP practice:
Population registered at each GP practice and population registered with various diseases, including obesity. (https://stats.gov.wales/en-GB/da26395b-945e-4fad-b886-00cbb565a14a/)

Outpatient referrals (April 2012 onwards):
A file for data filtered to trauma & orthopaedics and a file for metadata. (https://stats.gov.wales/en-GB/9e8f6f1d-d4aa-457a-a675-cc0e2caef3a5/)

NHS waiting list pathways (September 2011 onwards):
A file for waiting list data (26 weeks, 36 weeks, one year and two years) filtered to trauma & orthopaedics and a file for metadata. (https://stats.gov.wales/en-GB/27c78e17-6c03-4f50-90b6-21f8ce2ef27b/filtered/8s13e29uxtjk)
